Memory mock

Add mock data in ViewController.m's viewDidLoad:

- (void)viewDidLoad {
    [super viewDidLoad];
    // Do any additional setup after loading the view, typically from a nib.
    [self setPlist:@"main"];

    id data = @{@"name": @"My Pbind"};
    [self setData:data];
}

Data binding

Configure the data binding for the text property of main.tableview.plist:

KeyTypeValue
textString%(Hello %@!),$name

This expression if for string formating which would be parsed as:

[PBString stringWithFormat:@"Hello %@!", [controller.view.data valueForKey:@"name"]]
//              %           (Hello %@!), $name

Rerun the application then you will see the text changed as: "Hello My Pbind!".

More details of formating expression can refer to String format.

JSON mock

In last section we add the PBLocalhost directory by pbind watch, and now let's add some JSON files to mock server response.

Run the command in project directory:

pbind mock PBClient heroes

And we will got a PBClient/heroes.json under PBLocalhost. Let's add a key value pair:

{
  "name": "Mock Pbind"
}

then configure the clients in main.tableview.plist related to PBClient/heroes:

KeyTypeValue
clientsArray(1 item)
Item 0Dictionary(2 items)
clazzStringPBClient
actionStringheroes

Client is one of Pbind base component, used to fetch data. PBClient is the base class, you can customize your HTTPClient, DatabaseClient and so on by your business.

Here we design as clients array is because a View maybe related to multiable APIs. By this we can fetch all the datas in parallelly.

With the above configuration, Pbind will try to read the PBLocalhost/PBClient/heroes.json data as response in Debug mode for Simulator.

Now we can comment the memory mocking:

- (void)viewDidLoad {
    [super viewDidLoad];
    // Do any additional setup after loading the view, typically from a nib.
    [self setPlist:@"main"];

//    id data = @{@"name": @"My Pbind"};
//    [self setData:data];
}

Rerun the application, we will see: "Hello Mock Pbind!".

Try to make some changes of heroes.json and save:

{
  "name": "Mock Pbind, Good"
}

The page should be reloaded instantly as: "Hello Mock Pbind, Good!".
