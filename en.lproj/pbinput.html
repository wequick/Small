PBInput

PBInput extends from UITextField and add some properties for ease of configurating:

type

The type of the input with what we create a different input view for it. Accept types:

type input view note
text text keyboard
password text keyboard displays '*' on input
number numeric keyborad 0-9
decimal numeric keyboard 0-9 and '.'
phone phone keyboard 0-9 and '*', '+', '-', ';'
url url keyboard
email email keyboard '@'
date UIDatePicker 3 columns with year, month, day
time UIDatePicker 3 columns with hour, minute, second
datetime UIDatePicker 6 columns with year, month, day, hour, minute, second
month UIDatePicker 2 columns with year and month
select PBOptionPicker 1 column with option value

name

The name of the input used to be composed of a <name, value> pair.

value

The value of the input used to be composed of a <name, value> pair.

text

The text of the input for displaying.

min and max

The value limit of the input, accept values for different types:

type min/max e.g.
number integer 20
decimal float 20.0
date date components joined by '-' 2015-2-1
month month components joined by '-' 2015-2

unit

Default is nil. If set will place an UILabel with the unit string as the rightView of the input.

unitColor

The text color for the unit label.

options

Required by select input, format with NSArray<NSDictionary<NSString *, id> *>, as example:

[{text:'a', value:0}, {text:'b', value:1}]

The key text and value is required, text for displaying and value for model.

selector

The input view class name(should be subclass of PBInputView) for the select input, optional.

format

The format for the input text, accept formats for different typss:

type support format e.g.
number string format for the number "%lld piece(s)", 20 => "20 piece(s)"
decimal string format for the decimal "%.1lf", 20 => "20.0"
any sharp format for the text "## ## ##", 123456 => "12 34 56"

placeholderColor

The text color for the placeholder label of the input.

selectedTextColor

The text color for the selected state.

hidesCursor

Do not show the cursor for the input, default is NO.

maxlength

The maximum length of the input text. Calculated by [NSString length].

maxchars

The maximum character length of the input text, char as 1, unicode as 2. Calculated by [NSString charaterLength] (categorized by Pbind).

pattern

The regexp pattern for each character of the input text.

e.g. '[0-9]' will do nothing if you typed an 'a'.

validators

An array of dictionary used to validate for the whole input text. Each dictionary requires key of pattern and tips, the pattern is a regexp used to match the text and the tips refer to the error tips while the pattern mismatched.

e.g. [{pattern: '^S\w?$', tips: 'Should starts with S'}] will display 'Should starts with S' while you input a non-S-initial word.
