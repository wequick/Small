Client Component

The client component is implemented by PBClient, used to define the request standard and make it configurable for API sending.

Custom HTTPClient

This example show you how to create a custom Client.

Firstly we create a class extends PBClient and names as HTTPClient.

Then implement the loadRequest:success:failure: method in the HTTPClient.m:

@implementation HTTPClient

@pbclient(@"")
- (void)loadRequest:(PBRequest *)request success:(void (^)(id))success failure:(void (^)(NSError *))failure {

}

@end

Thereinto, we use @pbclient to register the Client alias, note that the empty string @"" will register the default client that means we will take it as default if we doesn't configure the client in Plist.

Here we just mock the server in memory:

static NSMutableArray *HEROES;

- (instancetype)init {
    if (self = [super init]) {
        HEROES = [[NSMutableArray alloc] init];
        [self addHeroWithName:@"Mr. Nice" desc:@"Nice"];
        [self addHeroWithName:@"Mr. Nice" desc:@"Nice"];
        [self addHeroWithName:@"Mr. Nice" desc:@"Nice"];
    }
    return self;
}

- (NSMutableDictionary *)heroWithName:(NSString *)name desc:(NSString *)desc {
    NSMutableDictionary *hero = [[NSMutableDictionary alloc] init];
    hero[@"name"] = name;
    hero[@"desc"] = desc;
    return hero;
}

- (void)addHeroWithName:(NSString *)name desc:(NSString *)desc {
    id hero = [self heroWithName:name desc:desc];
    [HEROES addObject:hero];
}

For API:

GET heroes

we return the HEROES:

@pbclient(@"")
- (void)loadRequest:(PBRequest *)request success:(void (^)(id))success failure:(void (^)(NSError *))failure {
    if ([request.method isEqualToString:@"GET"]) {
        if ([request.action isEqualToString:@"heroes"]) {
            success(HEROES);
        }
    }
}
